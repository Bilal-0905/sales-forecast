# Sales Forecast Tool

A Python machine learning tool that analyzes sales data and predicts future revenue using Linear Regression.

## Features
- Loads and analyzes monthly sales data
- Trains a machine learning model on historical data
- Predicts sales for future months
- Generates a chart showing actual vs predicted sales

## Libraries Used
- pandas
- matplotlib
- scikit-learn
- numpy
